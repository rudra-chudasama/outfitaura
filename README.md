# OutfitAura 👗  
An E-commerce website with an admin dashboard for managing products and categories.

---

## 🛍️ Overview
OutfitAura is a PHP & MySQL-based web project built for college submission and portfolio display.  
It provides both **customer features** (browsing and viewing products) and **admin features** (adding, editing, deleting products & categories).

---

## ✨ Features

### 🧾 Customer Side
- Browse products by category (Men, Women, Kids)
- Product details, price, and description
- Add to cart & basic checkout
- Responsive design for all devices

### ⚙️ Admin Panel
- Secure admin login
- Manage categories (add/edit/delete)
- Manage products with images, prices, stock
- Dashboard view of total products & categories

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Local Server:** XAMPP  

---

2. Move the project folder to  
   `C:\xampp\htdocs\outfitaura`
